TheBoysBot (The Misfits Private Discord Server)

This bot was developed as a revamp to a private server.

How this bot was built

This bot was built with the Red Discord Bot API. It currently is a node studio that, with "cogs" from git repositories, you can add different functions to your bot.
Red Discord Bot is an easy way to make a bot, especially for free!

How this bot is hosted 24/7

TheBoysBot is hosted on our local server. It auto-deploys when we update using a webhook and without worry of issues.

Bot Commands (Prefix = -)

Bot commands are listed below.

Commands that are requirements to know

-help - DMs command author (you!) all commands. Note: This will not work if you have DMs from non-friends off.
-netspeed - Shows bot speed from the server
-invite - Shows private bot invite link
-ping - Pings the bot to send back "Pong." (generally for testing)
-uptime - Shows bots current uptime.
-invite - Displays bot's oAuth invite. This link will not work for anyone other then the instance owner (which can be found by running -info.)
-info - Displays information about the bots environment.

Audio Tools for Music

We're working on more instances of this audio bot. We'll add more bots soon for multi-streaming!

-play [link/keywords] - Plays a song from YouTube (is capable of playing Spotify as well)
-autoplay - Begins autoplay.
-bump [track #] - Bumps a track to the top of the queue.
-bumpplay - Force play a URL or search for a track.
-disconnect - Disconnects the bot from the channel.
-eq - Equalizer Management
-genre - PIck a Spotify playlist from a list of categories
-now - Now playing
-pause - Pauses current track
-percent - Displays queue percentage.
-prev - Plays previous track
-queue - Displays song queue
-remove [track #] - Remove the specified track number.
-repeat - Toggles looping
-search - Find a track with Search
-seek [time] - Seek ahead in a track by seconds or to a specific time.
-shuffle - Toggles shuffle
-sing - Make TheBoysBot Sing!
-skip - Skips current track
-stop - Stops current track and clears queue.
-summon - Summons the bot to a VC
-volume [1%-150%] - Sets volume

Game Tools

If you'd like more games to play with TBB, feel free to brows Red's evergrowing Index of cogs! If you see any you'd like, use -contact to message the owner!

-audiotrivia - Starts an AudioTrivia Game (must be in a VC)
-trivia - Starts a Trivia game.
-dare - Dares a player
-neverhaveiever - Displays a Never Have I Ever question
-truth - Ask a truth question
-wouldyourather - Displays a Would You Rather question

Cards Against Humanity commands can only be ran via DMs with the bot.

-cahgames - Displays Cards Against Humanity games in progress.
-chat - Broadcasts a message to fellow CAH players.
-flushhand - Flushes the cards in your hand
-game - Displays the current game status
-hand - Shows your hand
-joincah - Join a CAH Game
-laid - Shows who's laid their cards
-lay - Lay your card(s) from your hand
-leavecah - Leaves the current game.
-newcah - Starts a new Cards Against Humanity game
-pick - As the judge - Pick the winning card(s)
-score - Displays the current score
