TheBoysBot (The Misfits Private Discord Server)

This bot was developed as a revamp to a private server.

How this bot was built

This bot was built with Discord Bot Studio from Steam. It currently is a node studio that, with the connecting of nodes, you can code bots.
While limited, there are other ways we counter all the limitations. They offer a wide range of mods to make implementing certain commands easier.

How this bot is hosted 24/7

TheBoysBot is hosted on a Heroku server via a Git repository. It auto-deploys when we update using a webhook and without worry of issues.
