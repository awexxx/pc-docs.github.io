The Forest Grove Pixelmon Server

The Forest Grove Pixelmon Server, Owned by Moonqii, developed by Alex Frantz.
This server uses Discord's API and a custom bot to send Discord messages to Minecraft and Minecraft messages to Discord.
This server also integrates with the PC Database to store linked accounts, giving certain people access to different commands, from Discord.

TheBoysBot - Discord

This bot is a private bot developed by Alex Frantz. It is used in a private server, with many commands such as setting game codes for game nights, movie nights, etc.
