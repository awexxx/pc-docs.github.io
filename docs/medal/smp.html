LuckPerms Admin Directions

In order to have tags for guilds, the server uses LuckPerms, a permissions manager.
Admins have access to the Web editor to create guild groups for people who ask to create a new guild. This doc will go over how to do that.

How to get a web editor link

All admins have command permissions in order to access the web editor. All you will need to do is run /lp editor in Minecraft, click the link, and you'll be at the editor.

How to add and color a group

Upon creating a group, you will be greeted with the following screen.
In the Group Name section, put the name of the guild (ex buildersguild, medalpog). In Display Name, feel free to put the same thing.
In Prefix, put the following; [(MC Color Code)Name of Faction&f]
Leave Prefix blank (for now, we can probably add them later)

Minecraft Color Codes

Adding a group to a user

As long as said user(s) have logged into the server and are active on it, you should see them pop up in the Players category.
You can remove the Default (Players) group by clicking the X, then pressing the + and then the group you made.

This can be accomplished via command by running /lp user (username) parent set (group name)

Saving changes

When you're done in the editor, click the green Save button in the top right of the window. It will give you a command to paste in MC, once you paste it, you should be finished!
